# Класи та об'єкти(ООП)

## Створення Классу

Ви вже знаєте:  Функції, If, Цикли...
Цей урок буде присвяцений Классам.

==Класс== - це...

Давайте розглянемо приклад:

???+ note "main.py"

    ``` mermaid

    graph TD

        A["**Предмет**<br/>(Має масу та об'єм)"] 
        
        A --> B["**Живі**<br/>(Дихають, ростуть, живляться)"]
        A --> C["**Неживі**<br/>(Не дихають, не ростуть)"]
        
        B --> D["**Тварини**<br/>(Активно рухаються)"]
        B --> E["**Рослини**<br/>(Виробляють кисень)"]
        
        D --> D1["**Кошеня**<br/>(Муркоче, грається)"]
        E --> E1["**Квітка**<br/>(Цвіте, пахне)"]
        
        C --> C1["**Стіл**<br/>(Міцний, дерев'яний)"]
    ```

В нашому випадку всі блоки це ==класи== та кожен клас має ^^ім'я^^.
Також ==класи== мають різні властивості. Та всі класи які походять від інших класів(тобто мають схожі властивосі) беруть від своїю батьків властивості та мають свої.


Щоб створити свій клас, треба:

1.Використовувати ключове слово ==class==

2.Створити ім'я для класу

3.Додавати функції і таке інше

Наведу простий приклад:

``` py title="oop.py"
--8<-- "oop.py:class1"
```

Як ми бачимо використовується ключове слово ==^^class^^==

???+ tip
    ==Класи завжди пишуться з великої літери==

Важливо пам'ятати про те що отступи в ==Python== дуже важливі

Тому пам'ятайте:

???+ failure
    ``` py
    class Subject():
    pass

    ```

???+ success

    ``` py
    --8<-- "oop.py:class1"
    ```
