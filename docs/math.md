# Математичні дії в Python

Python можна використовувати як _калькулятор_,наприклад:

```python
print(3+2)
```

Виведе: `5`

Ось основні для дій оператори та як вони використовуються:

| Вигляд | Опис               |
| ------ | ------------------ |
| `+`    | додавання          |
| `-`    | віднімання         |
| `*`    | множення           |
| `/`    | _звичайне_ ділення |

### Додавання `+`

```python
#1.
print(3+4)
#2.
print(5+6)
#3.
print(100+2)
```

### Віднімання `-`

```python
#1.
print(40-35)
#2.
print(7-3)
#3.
print(100-99)
```

### Множення `*`

```python
#1.
print(3*4)
#2.
print(3*2)
#3.
print(100*0)
```

### Ділення `/`

```python
#1.
print(4/2)
#2.
print(40/40)
#3.
print(33/3)
```

## Приклади на _багато_ дій

В математиці коли нам треба вирішити приклад на багато дій, ми робиро за певною послідовністю:

1.Якщо є дужки то робимо що в них

2.Піднесення в степінь(про це пізніше)

2-2.Дії на ділення та множнння

3.Додавання та віднімання

в **python** так само

```python

print(100/(46+4)-1)
```

Виведе: `1`

Тобто python виконує приклади за правилами як в **математиці**

### Python дії +

Крім звичайних дії є ще декілька додаткових:

| Вигляд | Опис                   |
| ------ | ---------------------- |
| `**`   | піднесення до степення |
| `%`    | остача від ділення     |
| `//`   | цілочисельне ділення   |

### Піднесення до степення

Воно нічим не відрізняється від того, шо в математиці

```python

print(3**2)
```

Виведе: `9`

### Остача від ділення

Це залишок(що не вийшло поділити)

```python

print(20%11)
```

Виведе: `9`

### Цілочисельне ділення

Це скільки **цілих** разів дільник 'помістився' в ділене

```python

print(11//3)
```

Виведе: `3`
