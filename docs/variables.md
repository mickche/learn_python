# Змінні

## **Змінна — це ім'я, яке ми даємо якомусь значенню, щоб комп'ютер його запам'ятав**

Змінні можкуь мати як дані так і якісь вирази:

```python
a = 10
```

В нашому коді ми створюєма змінну з ім'ям ==**a**==, та привласнюємо значення ==**10**==

### Правилання написання змінних

Якщо не дотримуватися цих правил то будуть помилки від python:

1.Змінні не можуть починатись з _цифр_

2.Не можуть містити пробіли

3.Та не можуть містити символи(окрім `_`)

!!! tip

    Змінні можуть мати **будь яку** назву, крім тих які не підтримуються правилами

## Типи даних

Перед тим як ми продовжемо, хотів розповісти про **основні** типи даних:

| Назва | Опис                        |
| ----- | --------------------------- |
| int   | ціле число(1,2,3...876...)  |
| float | дробове число(1.05,24.7...) |
| str   | текст('hello',"null"...)    |
| bool  | істина чи ні(True,False)    |

> В Python текст можна писати як в `''` так і в `""`

> В Python True та False пишуться з **великої літери**

### Повертаємось до змінних:

```python

a = 1+4

print(a)
```

Як я вже казав в змінних можна записувати якісь дії.
Так от, коли ми надрукуємо **a**, то буде 5.

### Вивід тексту зі змінними

Для того щоб вивести привітання комусь, ми використовуємо ім'я та вік

Але якщо ми зробимо так:

```python

    age = 14

    name = 'Grey'

    print('Hello name, you are age, right?')

```

То це **не працюватиме**

Правильно так:

```python

    print('Hello',name,'you are,',age,'right?')

```

Пізніше буде більш _красивий_ варіант
