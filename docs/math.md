# Математичні дії

Коли ми хочемо вирішити задачку по фізици відставивши числа, ми можеио використовувати дії в стовпчик або калькулятор.
В **Python** можна вирішувати приклади(як на калькуляторі), наприклад 25 помножити на 100:

```py
print(25*100)
```

Виведе: 2500

!!! tip

    В **Python** операція *множення* пишется через `*`

Давайте зробимо більш корисну операцію
