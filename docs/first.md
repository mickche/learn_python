# Ваша перша програма

Щоб почати програмування, давайте створимо вашу першу програму:

1. Відкриваємо редактор коду (редактори коду тут `click`).
2. Створюємо файл **1.py** (або з будь-якою іншою назвою).

!!! tip

    Всі файли для **Python** пишуться з розширенням `.py`

3. Вставляємо цей код:

```python

print('Hello World with cat and Python')
```

4. Запускаємо.

Має бути такий результат:

```text
Hello World with cat and Python
```
