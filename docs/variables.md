# Змінні

## **Змінна — це ім'я, яке ми даємо якомусь значенню, щоб комп'ютер його запам'ятав**

Змінні можкуь мати як дані так і якісь вирази:

```python
a = 10
```

В нашому коді ми створюєма змінну з ім'ям ==**a**==, та привласнюємо значення ==**10**==

## Типи даних

Перед тим як ми продовжемо, хотів розповісти про **основні** типи даних:

| name  | description                 |
| ----- | --------------------------- |
| int   | ціле число(1,2,3...876...)  |
| float | дробове число(1.05,24.7...) |
| str   | текст('hello',"null"...)    |
| bool  | істина чи ні(True,False)    |

> В Python текст можна писпти як в ==''== так і в ==""==

> В Python True та False пишуться з ==великої літери==

### Повертаємось до змінних:

```
a = 1+4
```

Як я вже казав в змінних можна записувати якісь дії.
Так от, коли ми надрукуємо **a**, то буде 5(Python може вирішувати приклади).
