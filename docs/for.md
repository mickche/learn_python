# Цикли

 Цикли - інструкція, яка змушує програму повторювати певний блок коду, доки виконується задана умова, або певну кількість разів.


### Синтаксис

```
for назва_змінної in range(...):
    ...
```

## Як працює функція range()

Функція range() задає кількість повторень і може приймати від одного до трьох параметрів:


1.range(stop) — повторює код від 0 до stop - 1.


Приклад: range(5) згенерує числа: 0, 1, 2, 3, 4 (всього 5 разів).


2.range(start, stop) — починає з start і йде до stop - 1.


Приклад: range(1, 5) згенерує числа: 1, 2, 3, 4.


3.range(start, stop, step) — йде від start до stop - 1 з кроком step.


Приклад: range(1, 10, 2) згенерує непарні числа: 1, 3, 5, 7, 9.


### Робота зі списками (Ітерація по колекції)

Цикл for в Python ідеально підходить для перебору елементів у списках (масивах). Замість range() ми підставляємо сам список, і змінна по черзі набуває значення кожного елемента.
Python

#### Створюємо список фруктів
fruits = ["яблуко", "банан", "апельсин"]

## Перебираємо елементи списку
for fruit in fruits:
    print(f"Я люблю їсти {fruit}")

Результат роботи:
Plaintext

Я люблю їсти яблуко
Я люблю їсти банан
Я люблю їсти апельсин

> Для зручності назву списку зазвичай роблять у множині (fruits), а назву змінної циклу — в однині (fruit).



## Цикл While

Цикл while (з англ. «поки») — це керуюча конструкція, яка виконує блок коду (тіло циклу) доти, доки певна умова залишається істинною (True).

1. Синтаксис (на прикладі Python)
Python

while умова:
# тіло циклу (код, який виконується)
# зміна змінної (крок циклу)

2. Як це працює (Алгоритм)

Програма перевіряє умову.

Якщо умова істинна (True), виконується код усередині циклу.

Після виконання тіла циклу програма повертається до перевірки умови.

Якщо умова хибна (False), цикл завершується, і програма виконується далі.

3. Базовий приклад

Виведемо числа від 1 до 5:

``` python

counter = 1

while counter <= 5:  # 2. Умова
print(counter)
counter += 1  # 3. Крок збільшуємо, бо цикл буде нескінченним!
```

4. Важливі оператори для керування циклом

break — достроково перериває цикл і виходить із нього.

continue — пропускає решту коду в поточній ітерації та переходить до наступної перевірки умови.

Приклад із break:

``` python

while True:
answer = input("Введіть 'вихід' для завершення: ")
if answer == 'вихід':
break
```
5. Головна небезпека при роботі з while

!!! tip
    Нескінченний цикл: Якщо умова завжди залишається True (наприклад, ви забули змінити змінну лічильника всередині циклу), програма «зависне».

Приклад помилки:

``` python
i = 1
while i <= 5:
print(i)  # Змінна i не змінюється, цикл працюватиме вічно.
```
6. Коли використовувати while, а коли for?

Цикл for використовують, коли кількість повторень відома заздалегідь (наприклад, пройти по елементах списку або повторити рівно 10 разів).

Цикл while використовують, коли кількість повторень невідома і залежить від певних умов (наприклад, чекати, поки користувач введе правильний пароль, або поки не закінчиться файл).
