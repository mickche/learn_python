    1. Що таке список?

    Список — це впорядкована колекція елементів. Куди можна покласти що завгодно: числа, текст або навіть інші списки.

    • Список створюється за допомогою квадратних дужок [].

    • Елементи всередині розділяються комами.

        # Порожній список
    empty_list = []

    # Список чисел
    numbers = [10, 20, 30, 40]
    Tip

    Щоб створити пустий список, можна використати list()

    2. Індекси

    Кожен елемент у списку має свій "номер" — індекс.

    fruits = ["яблуко", "банан", "апельсин"]
#            0         1         2

# Отримуємо елемент за індексом:
print(fruits[0]) # яблуко
print(fruits[2]) # апельсин
    Tip

    В python індексація починається з 0

    Зворотна нумерація

    Якщо потрібно взяти елементи з кінця списку, використовують мінусові індекси. Останній елемент — це завжди [-1]

    print(fruits[-1])  # апельсин
print(fruits[-2])  # банан

    3. Зміна елементів списку

    Списки є змінними. Це означає, що ви можете легко замінити будь-який елемент.

    4. Основні методи cписків

    • append()
    • remove()
    • index()

    Метод append()

    Метод append() додає нові елементи до списку:

    my_list = []
my_list.append('apple')

    Метод remove()

    Метод remove() видаляє елементи зі списку: 

    my_list = ['apple','golden_apple', 'gold']
my_list.remove('gold')

    Метод index()

    Метод inde() повертає індекс цього елемента(якщо він є) 

    my_list = ['apple','golden_apple']
my_list.index('apple') # буде 0
